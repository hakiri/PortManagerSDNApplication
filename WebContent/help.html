  1. SDN Port Manager Application FAQ

Why do we need SDN Port Manager Application?

There is a common problem in campus or enterprise networks, where IT administrators want to restrict usage of ethernet ports to specific users or mac addresses or ip addresses. This helps IT administrators control who accesses the network and from where in thier wired network.

Now to accomplish this in a multi-vendor environment, either admin's have to go around to configure each of thier ede or accesses switches for correct ACL's or policies of some sort. There is no way possible today, where admin's can create and apply such a policy from a centralized place in a multi-vendor setup.

This problem can be easily solved via SDN Controlled network using SDN Port Manager.

What is SDN Port Manager Application?

SDN Port Manager Application is used to allow or deny openflow enabled physical ports of a device for supplied mac or ip addresses.
